# Calculadora CLI

Mini proyecto de calculadora por consola usando Python.

## Qué practica

- funciones;
- entrada de datos con `input`;
- condicionales;
- manejo básico de errores;
- bucle `while`.

## Ejecutar

Desde la raíz del repositorio:

```bash
python proyectos/consola/calculadora_cli/main.py
```

O dentro de esta carpeta:

```bash
python main.py
```
