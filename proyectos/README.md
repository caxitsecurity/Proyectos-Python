# Proyectos Python

Carpeta para proyectos más completos y scripts útiles.

## Estructura sugerida

```text
proyectos/
├── automatizacion/
├── consola/
├── archivos/
└── ciberseguridad-labs/
```

## Ideas de proyectos

| Proyecto | Descripción |
|---|---|
| Calculadora CLI | Calculadora simple por terminal |
| Organizador de archivos | Ordena archivos por extensión o fecha |
| Generador de contraseñas | Crea contraseñas aleatorias seguras |
| Lector de CSV | Lee y resume archivos `.csv` |
| Monitor de sistema | Muestra CPU, RAM o disco desde terminal |
| Scanner de puertos lab | Solo para máquinas propias o laboratorios legales |

## Plantilla recomendada para cada proyecto

```text
nombre-proyecto/
├── main.py
├── README.md
└── requirements.txt
```

Cada proyecto debería explicar:

- qué hace;
- cómo se ejecuta;
- qué aprendí;
- posibles mejoras futuras.
